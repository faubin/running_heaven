About the Project

Running Heaven is a web application which suggest the most pleasant running routes given a starting point, an end point, and a target distance. It considerers the street intersections, the parks, and the trees. These features can be turned off if desired. To obtain a loop, enter two nearby intersections, or use the random end point generator with half the intended running distance and run the route forth and back.

The Python code for this application is available on Github.com. The data is downloaded from the NYC Open Data website. Addtional information about how running routes are optimized is available here.
